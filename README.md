## College Board Science Practices Covered

1. **Creating Representations** — Diagrams, graphs, FBDs, circuit schematics
2. **Mathematical Routines** — Algebraic & calculus derivations, dimensional analysis
3. **Scientific Argumentation** — Qualitative reasoning, claim-evidence-reasoning
4. **Experimental Design** — Lab setups, data collection, error analysis
5. **Qualitative/Quantitative Translation** — Connecting physical meaning to math

## How to Use This Database

Each unit file follows this exact schema:
- **Part A:** Chapter Blueprint & Concepts (3–5 sub-chapters)
- **Part B:** Diagram System (Mermaid.js, ASCII FBDs, graph specs)
- **Part C:** 20 Worked Examples with step-by-step solutions
- **Part D:** 50-Question Test Bank + 10 FRQs + Answer Key Matrix
